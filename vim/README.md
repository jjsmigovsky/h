# VIMRC

## Setup pathogen
```
mkdir -p ~/.vim/autoload ~/.vim/bundle

curl -Sso ~/.vim/autoload/pathogen.vim https://raw.github.com/tpope/vim-pathogen/master/autoload/pathogen.vim

```


## Add some plugins
```
cd ~/.vim/bundle

git clone https://github.com/hashivim/vim-hashicorp-tools.git

```


## Move the .vimrc
```
mv vimrc ~/.vimrc
```
